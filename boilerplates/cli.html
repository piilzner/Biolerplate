pbp CLI

cli tool for installing pbp boilerplates

Fork me on Github

Installation

$ git clone https://github.com/piilzner/pbpcli.git

$ cd pbpcli

$ npm install

$ npm link

Usage

$ pbp --help

$ pbp install <boilerplate> <directory name>

$ pbp list

$ pbp generate <type> <file name>
